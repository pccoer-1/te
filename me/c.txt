### üß© **IF-ELSE & LOOP**

```sql

  v_marks INT := 85;
BEGIN
  IF v_marks>=75 THEN
    DBMS_OUTPUT.PUT_LINE('Distinction');
  ELSE
    DBMS_OUTPUT.PUT_LINE('Pass');
  END IF;
END;
/
```

### üß© **Cursor Example**

```sql
DECLARE
  CURSOR c IS SELECT * FROM student;
  v_row student%ROWTYPE;
BEGIN
  OPEN c;
  LOOP
    FETCH c INTO v_row;
    EXIT WHEN c%NOTFOUND;
    DBMS_OUTPUT.PUT_LINE(v_row.name);
  END LOOP;
  CLOSE c;
END;
/
```


### üß© **Insert Documents**


### üß© **Query Documents**

```javascript
db.student.find();
db.student.find({city:"Pune"});
db.student.find({marks:{$gt:80}});
db.student.find({}, {name:1,city:1,_id:0});
```

### üß© **Update & Delete**

```javascript
db.student.updateOne({id:1},{$set:{marks:95}});
db.student.updateMany({city:"Pune"},{$inc:{marks:5}});
db.student.deleteOne({id:2});
db.student.deleteMany({city:"Delhi"});
```

### üß© **Aggregation**

```javascript
db.student.aggregate([
  {$group:{_id:"$city",avgMarks:{$avg:"$marks"}}},
  {$sort:{avgMarks:-1}}
]);
```

### üß© **Indexes**

```javascript
db.student.createIndex({name:1});
db.student.getIndexes();
```

